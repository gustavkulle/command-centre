Connect Google Sheets

One-time setup to automatically send your daily data to a Google Sheet. Takes about 5 minutes.

1
Go to sheets.google.com → create a new sheet called "Command Centre Data"
2
Click Extensions → Apps Script → delete all existing code → paste the Apps Script file I gave you → Save
3
Click Deploy → New Deployment → Web app → Execute as: Me → Who has access: Anyone → Deploy → Authorize → Allow
4
Copy the Web App URL and paste it below
Your data stays private. The webhook only accepts POST requests from this dashboard and writes to your own Google Sheet.
